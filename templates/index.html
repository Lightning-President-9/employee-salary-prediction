Employee Salary Prediction

Machine learning–based income classification dashboard

About This Project

This project focuses on predicting employee salary categories using demographic and employment attributes from the Adult Census Income dataset. A machine learning classification model is used along with interactive data visualizations for exploratory analysis.

  • Dataset (Kaggle): Adult Census Income Dataset
  • Model Notebook: Training & Analysis Notebook
  • Base GitHub Repository: Employee Salary Prediction (Main Repo)
  • Deployed Flask Repository: Flask Deployment Repository

⚠ Disclaimer: This project is for educational and demonstrational purposes only. Predictions are probabilistic and must not be used for employment or compensation decisions.
